# Moji - Real-time Chat & Social Platform (Frontend)
## 🚀 Tính năng cốt lõi (Core Features)

*   **Real-time Messaging:** Truyền tải tin nhắn tức thời hai chiều sử dụng kết nối WebSockets (Socket.io).
*   **Message Status Tracking:** Cập nhật trực tiếp trạng thái của tin nhắn theo thời gian thực 
*   **User Presence:** Hiển thị trạng thái hoạt động động của người dùng (Online/Offline)
*   **Secure Authentication:** Luồng đăng nhập bảo mật tích hợp mã hóa Token dựa trên JWT, xử lý chuyển trang mượt mà.
*   **Robust Validation:** Xác thực cấu trúc dữ liệu chặt chẽ ngay tại client-side cho toàn bộ hệ thống biểu mẫu (Form handling).

## 🛠️ Công nghệ sử dụng (Tech Stack)

*   **Core:** React
*   **Language:** TypeScript
*   **State Management:** Zustand 
*   **Real-time Communication:** Socket.io-client
*   **Form & Schema Validation:** Zod
*   **Styling:** Tailwind CSS
